## Consult the Documentation

**Before modifying or extending hoist-react code, use the Hoist developer tools** (MCP server or
CLI) to search and read the framework documentation. See the "Hoist Developer Tools and
Documentation" section in AGENTS.md for tool usage. The package READMEs and concept docs are the
authoritative reference for how Hoist works -- skipping them risks producing code that conflicts
with established patterns or misses built-in functionality.
